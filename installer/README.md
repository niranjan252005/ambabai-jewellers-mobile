# Jewelry Mobile App - Complete Package 
 
## What's Included 
- Complete Flutter mobile app source code 
- Node.js backend server 
- SQLite database with sample data 
- Custom diamond branding throughout 
- Photo upload functionality 
- Share functionality for WhatsApp and social media 
- Complete documentation 
 
## Quick Start 
1. Run setup.bat (Windows) or setup.sh (macOS/Linux) 
2. Install Flutter SDK from https://flutter.dev 
3. Run build-mobile.bat to create installable apps
